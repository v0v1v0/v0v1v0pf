fqa_citations R Documentation

Citations for Floristic Quality Assessment Databases

Description

A data set containing citations for the regional Floristic Quality Assessment (FQA) databases contained in this package, as well as information on approval status, and notes regarding the proper application of the regional database.

Usage

fqa_citations

Format

A data frame

fqa_db

Regional FQA database

recommended

Indicates if the regional FQA database was recommended for use by the U.S. Army Corps of Engineers

notes

Notes on the limitations or recommended usage of the regional FQA database

citation

A citation for the regional FQA database
