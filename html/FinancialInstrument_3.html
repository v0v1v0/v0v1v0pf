Extract a single row from each day in an xts object

Description

Extract a single row from each day in an xts object

Usage

.to_daily(x, EOD_time = "15:00:00")

Arguments

x

xts object of sub-daily data.
EOD_time

time of day to use.

Value

xts object with daily scale.
