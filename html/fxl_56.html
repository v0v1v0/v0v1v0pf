scr_yoverride R Documentation

yoverride

Description

Override the y axis (or axes) limits

Usage

scr_yoverride(
  core_frame,
  var,
  ydelta = 1,
  yticks = NULL,
  ydraws = NULL,
  ytickslabs = NULL
)

Arguments

core_frame

fxl object
var

from base
ydelta

skips between ticks (can override)
yticks

tick values (numerical)
ydraws

specify axes manual
ytickslabs

tick labels

Value

nothing, run for side effects
