date_yq R Documentation

date_yq

Description

creates a quarterly date vector from a date vector

Usage

date_yq(x)

Arguments

x

a date

Value

date vector

Examples


seq.Date(lubridate::ymd(20200101), lubridate::ymd(20220101), length.out = 10) -> d1
d1 %>%
  tibble::enframe() %>%
  dplyr::mutate(YQ = date_yq(value))
