icevision_Normalize R Documentation

Normalize

Description

Divide pixel values by 255 = 2**8 - 1, subtract mean per channel and divide by std per channel.

Usage

icevision_Normalize(
  mean = list(0.485, 0.456, 0.406),
  std = list(0.229, 0.224, 0.225),
  max_pixel_value = 255,
  always_apply = FALSE,
  p = 1
)
