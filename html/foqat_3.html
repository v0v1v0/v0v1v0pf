aqi R Documentation

Demo data of air quality

Description

5 days air quality data (1 min resolution) includes: NO, NO2, CO, SO2, O3. The variables are as follows:

Usage

aqi

Format

A data frame with 7140 rows and 6 variables:

Time

Time for data

NO

Nitric oxide (NO)

NO2

Nitrogen Dioxide (NO2)

CO

Carbon monoxide (CO)

SO2

Sulfur dioxide (SO2)

O3

Ozone (O3)
