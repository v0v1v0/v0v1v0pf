ResNet R Documentation

ResNet

Description

Base class for all neural network modules.

Usage

ResNet(
  block,
  layers,
  num_classes = 1000,
  zero_init_residual = FALSE,
  groups = 1,
  width_per_group = 64,
  replace_stride_with_dilation = NULL,
  norm_layer = NULL
)

Arguments

block

the blocks that need to passed to ResNet
layers

the layers to pass to ResNet
num_classes

the number of classes
zero_init_residual

logical, initializer
groups

the groups
width_per_group

the width per group
replace_stride_with_dilation

logical, replace stride with dilation
norm_layer

norm_layer
