define.bands

Description

define.bands defines the spectral regions for the Fluspect-B model

Usage

define.bands()

Details

Define spectral regions for SCOPE v_1.40
All spectral regions are defined here as row vectors
WV Jan. 2013

Value

a spectral object.

Examples
