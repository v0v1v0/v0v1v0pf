get_mtime R Documentation

Get the mtime Attribute from an Object

Description

We set modification times on some objects, this is a convenience wrappers to attr.

Usage

get_mtime(x)

Arguments

x

An object.

Value

The value of attr(attr(x, "path", "mtime").

See Also

Other file utilities: clipboard_path(), delete_trailing_blank_lines(), delete_trailing_whitespace(), develop_test(), file_copy(), file_modified_last(), file_save(), find_files(), get_lines_between_tags(), get_unique_string(), grep_file(), is_files_current(), is_path(), paths, search_files(), split_code_file(), touch()

Examples

x <- 2
path <- tempfile()
touch(path)
x <- set_path(x, path)
get_mtime(x)
