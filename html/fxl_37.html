scr_anno_guide_line

Description

This is an annotation illustrating an aim/reduction line

Usage

scr_anno_guide_line(
  core_frame,
  coords,
  facet = NA,
  color = "black",
  lty = 1,
  lwd = 1
)

Arguments

core_frame

fxl object
coords

start and finish coords for aim line
facet

panel to draw upon
color

from base
lty

line type
lwd

line width

Value

a layer to the core plotting object
