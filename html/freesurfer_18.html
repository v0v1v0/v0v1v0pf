freesurferdir R Documentation

Get Freesurfer's Directory

Description

Finds the FREESURFER_HOME from system environment or getOption("freesurfer.path") for location of Freesurfer functions and returns it

Usage

freesurferdir()

freesurfer_dir()

fs_dir()

Value

Character path

Examples

if (have_fs()) {
 freesurferdir()
 freesurfer_dir()
 fs_dir()
}
