load_dataset R Documentation

Load_dataset

Description

A helper function for getting a DataLoader for images in the folder 'test_path', with batch size 'bs', and number of workers 'num_workers'

Usage

load_dataset(test_path, bs = 4, num_workers = 4)

Arguments

test_path

test path (directory)
bs

batch size
num_workers

number of workers

Value

None
