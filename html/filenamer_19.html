tag R Documentation

Insert tag or extension and coerce to character

Description

This function inserts a tag or extension into a file name and returns a charcter vector.

Usage

tag(x, ...)

Arguments

x

a filename or character
...

arguments passed to insert

Value

a character vector

Examples

x <- "data.txt"
y <- tag(x, "qc")
print(y)
f <- as.filename(x)
g <- tag(f, "qc")
print(g)
