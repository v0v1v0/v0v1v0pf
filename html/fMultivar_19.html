utils-grid2d R Documentation

Bivariate Density Tools

Description

Grid generator for bivariate distributions.

Usage

grid2d(x = (0:10)/10, y = x)

Arguments

x, y

two numeric vectors defining the x and y coordinates.

Value

grid2d returns a list with two vectors named $x and $y spanning the grid defined by the coordinate vectors x and y.

Author(s)

Diethelm Wuertz.

Examples

## grid2d -
   # Create a square grid:
   x <- seq(0, 10, length = 6)
   X <- grid2d(x = x, y = x)
   cbind(X$x, X$y)
