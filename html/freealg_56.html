nterms R Documentation

Number of terms in a freealg object

Description

Number of terms in a freealg object; number of coefficients

Usage

nterms(x)

Arguments

x

Freealg object

Value

Returns a non-negative integer

Author(s)

Robin K. S. Hankin

Examples

(a <- freealg(list(1:3,4:7,1:10),1:3))
nterms(a)
nterms(a+1)
nterms(a*0)
