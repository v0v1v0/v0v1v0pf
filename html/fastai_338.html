HF_QABeforeBatchTransform

Description

Handles everything you need to assemble a mini-batch of inputs and targets, as well as decode the dictionary produced

Usage

HF_QABeforeBatchTransform(
  hf_arch,
  hf_tokenizer,
  max_length = NULL,
  padding = TRUE,
  truncation = TRUE,
  is_split_into_words = FALSE,
  n_tok_inps = 1,
  ...
)

Arguments

hf_arch

architecture
hf_tokenizer

tokenizer
max_length

maximum length
padding

padding or not
truncation

truncation or not
is_split_into_words

into split into words or not
n_tok_inps

number of tok inputs
...

additional arguments

Details

as a byproduct of the tokenization process in the 'encodes' method.

Value

None
