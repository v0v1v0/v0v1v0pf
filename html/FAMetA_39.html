Description

Search FA isotopes

Usage

searchFAisotopes(msbatch, dmz = 5, coelCutoff = 0.7)

Arguments

msbatch

annotated msbatch.
dmz

mz tolerance in ppm.
coelCutoff

coelution score threshold between parent and isotope peaks.

Value

fadata list.
