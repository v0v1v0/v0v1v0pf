norm R Documentation

Matrix norm.

Description

norm(X) returns the largest singular value of X; it is equivalent to svd(X, nu = 0, nv = 0)$d[1].

Usage

norm(X)

Arguments

X

a numeric matrix.

Value

The largest singular value of X.

Author(s)

Øyvind Langsrud and Bjørn-Helge Mevik
