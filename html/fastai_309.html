get_dls R Documentation

Get dls

Description

Given image files from two domains ('pathA', 'pathB'), create 'DataLoaders' object.

Usage

get_dls(
  pathA,
  pathB,
  num_A = NULL,
  num_B = NULL,
  load_size = 512,
  crop_size = 256,
  bs = 4,
  num_workers = 2
)

Arguments

pathA

path A (from domain)
pathB

path B (to domain)
num_A

subset of A data
num_B

subset of B data
load_size

load size
crop_size

crop size
bs

bathc size
num_workers

number of workers

Details

Loading and randomly cropped sizes of 'load_size' and 'crop_size' are set to defaults of 512 and 256. Batch size is specified by 'bs' (default=4).

Value

None
