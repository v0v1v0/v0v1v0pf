GatherPredsCallback R Documentation

GatherPredsCallback

Description

'Callback' that saves the predictions and targets, optionally 'with_loss'

Usage

GatherPredsCallback(
  with_input = FALSE,
  with_loss = FALSE,
  save_preds = NULL,
  save_targs = NULL,
  concat_dim = 0
)

Arguments

with_input

include inputs or not
with_loss

include loss or not
save_preds

save predictions
save_targs

save targets/actuals
concat_dim

concatenate dimensions

Value

None
