Difference in Sets

Description

Difference in Sets

Usage

S1 %-% S2

Arguments

S1

A Set
S2

A Set

Details

Both S1 and S2 must be Sets.

Value

Returns the difference S1 - S2.

Examples

# Build two sparse sets
S <- Set$new(attributes = c("A", "B", "C"))
S$assign(A = 1, B = 1)
T <- Set$new(attributes = c("A", "B", "C"))
T$assign(A = 1)

# Difference
S %-% T
