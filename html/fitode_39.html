Internal function for plotting methods

Usage

plot_internal(
  pred,
  data,
  onepage = TRUE,
  xlim,
  ylim,
  xlabs,
  ylabs,
  col.traj = "black",
  lty.traj = 1,
  col.conf = "black",
  lty.conf = 4,
  add = FALSE,
  ...
)

Arguments

pred

prediction objects
data

observed data
onepage

(logical) print all figures on one page?
xlim

x coordinates range
ylim

y coordinates range
xlabs

a label for the x axis
ylabs

a label for the y axis
col.traj

colour of the estimated trajectory
lty.traj

line type of the estimated trajectory
col.conf

colour of the confidence intervals
lty.conf

line type of the confidence intervals
add

add to another plot?
...

additional arguments to be passed on to the plot function
