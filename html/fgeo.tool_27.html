drop_if_na R Documentation

Drop if missing values.

Description

Valuable mostly for its warning.

Usage

drop_if_na(dfm, x)

Arguments

dfm

A dataframe.
x

String giving a column name of dfm.

Value

A dataframe.

See Also

tidyr::drop_na().

Examples

dfm <- data.frame(a = 1, b = NA)
drop_if_na(dfm, "b")
drop_if_na(dfm, "a")
