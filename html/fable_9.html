Extract estimated states from an ETS model.

Description

Extract estimated states from an ETS model.

Usage

## S3 method for class 'ETS'
components(object, ...)

Arguments

object

An estimated model.
...

Unused.

Value

A fabletools::dable() containing estimated states.

Examples

as_tsibble(USAccDeaths) %>%
  model(ets = ETS(log(value) ~ season("A"))) %>%
  components()
