lr_find R Documentation

Lr_find

Description

Launch a mock training to find a good learning rate, return lr_min, lr_steep if 'suggestions' is TRUE

Usage

lr_find(
  object,
  start_lr = 1e-07,
  end_lr = 10,
  num_it = 100,
  stop_div = TRUE,
  ...
)

Arguments

object

learner
start_lr

starting learning rate
end_lr

end learning rate
num_it

number of iterations
stop_div

stop div or not
...

additional arguments to pass

Value

data frame

Examples


## Not run: 

model %>% lr_find()
model %>% plot_lr_find(dpi = 200)


## End(Not run)
