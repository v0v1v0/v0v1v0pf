fslsqrt.help R Documentation

fslsqrt Help

Description

This function calls fslmaths's help, as fslsqrt is a wrapper for fslmaths

Usage

fslsqrt.help(...)

Arguments

...

passed to fslmaths.help

Value

Prints help output and returns output as character vector

Examples

if (have.fsl()){
 fslsqrt.help() 
}
