icevision_GridDropout R Documentation

GridDropout

Description

GridDropout, drops out rectangular regions of an image and the corresponding mask in a grid fashion.

Usage

icevision_GridDropout(
  ratio = 0.5,
  unit_size_min = NULL,
  unit_size_max = NULL,
  holes_number_x = NULL,
  holes_number_y = NULL,
  shift_x = 0,
  shift_y = 0,
  random_offset = FALSE,
  fill_value = 0,
  mask_fill_value = NULL,
  always_apply = FALSE,
  p = 0.5
