replaceRotTag

Description

insert html tags for rotating text

Usage

replaceRotTag(x, l, dims)

Arguments

x

return value of readLines(), HTML code containing data table
l

return values of spanTag()
dims

return value of dim()

Value

returns modified version of HTML code containing data table
