cartdepth R Documentation

Depth

Description

Return the dept of a decision tree.

Usage

cartdepth(model)

Arguments

model

The decision tree.

Value

The depth.

See Also

CART, cartinfo, cartleafs, cartnodes, cartplot

Examples

require (datasets)
data (iris)
model = CART (iris [, -5], iris [, 5])
cartdepth (model)
