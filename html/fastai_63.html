AudioBlock_from_folder R Documentation

AudioBlock from folder

Description

Build a 'AudioBlock' from a 'path' and caches some intermediary results

Usage

AudioBlock_from_folder(
  path,
  sample_rate = 16000,
  force_mono = TRUE,
  crop_signal_to = NULL
)

Arguments

path

directory, path
sample_rate

sample rate
force_mono

force mono or not
crop_signal_to

int, crop signal

Value

None
