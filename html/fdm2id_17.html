cartinfo R Documentation

CART information

Description

Return various information on a CART model.

Usage

cartinfo(model)

Arguments

model

The decision tree.

Value

Various information organized into a vector.

See Also

CART, cartdepth, cartleafs, cartnodes, cartplot

Examples

require (datasets)
data (iris)
model = CART (iris [, -5], iris [, 5])
cartinfo (model)
