ausbeer R Documentation

Quarterly Australian Beer production

Description

Total quarterly beer production in Australia (in megalitres) from 1956:Q1 to 2008:Q3.

Usage

data(ausbeer)

Format

Quarterly time series of class ts.

Source

Australian Bureau of Statistics. Cat. 8301.0.55.001.

Examples

data(ausbeer)
seasonplot(ausbeer)
