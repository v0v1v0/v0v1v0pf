download_flightsbr_file R Documentation

Download file from url

Description

Download file from url

Usage

download_flightsbr_file(
  file_url = parent.frame()$file_url,
  showProgress = parent.frame()$showProgress,
  dest_file = temp_local_file,
  cache = cache
)

Arguments

file_url

String. A url passed from get_flights_url.
showProgress

Logical, passed from read_flights
dest_file

String, passed from read_flights
cache

Logical, passed from read_flights

Value

Silently saves downloaded file to temp dir.

Examples

## Not run:  if (interactive()) {
# Generate url
file_url <- get_flights_url(type='basica', date=200011)

# download data
download_flightsbr_file(file_url=file_url,
                        showProgress=TRUE,
                        dest_file = tempfile(fileext = ".zip")
                       )
}
## End(Not run)
