Tidy a ATA model

Description

Tidy a ATA model

Usage

## S3 method for class 'ATA'
tidy(x, ...)

Arguments

x

An estimated model.
...

Unused.

Value

The model's coefficients in a 'tibble'.

Examples

library(fable.ata)
as_tsibble(USAccDeaths) %>%
  model(ata = AutoATA(value ~ trend("A") + season("M"))) %>% tidy()
