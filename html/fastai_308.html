get_dicom_files

Description

Get dicom files in 'path' recursively, only in 'folders', if specified.

Usage

get_dicom_files(path, recurse = TRUE, folders = NULL)

Arguments

path

path to files
recurse

recursive or not
folders

folder names

Value

lsit of files
