fpp-class R Documentation

Class "fpp"

Description

Objects of the fpp class are used by methods and functions of the fp package, to visualize and compute the fixed-point property in response time data.

Objects from the Class

Objects can be created by calls of the form new("fpp", ...).

Slots

dens:

Object of class "array". This is an array of objects of class "density".

diff:

Object of class "data.frame". This is a dataframe with the paired differences between densities.

dat:

Object of class "data.frame". This is a dataframe with the input data.

Methods
