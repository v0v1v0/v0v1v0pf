meco2phyloseq R Documentation

Transform 'microtable' object of 'microeco' package to the 'phyloseq' object of 'phyloseq' package <doi:10.1371/journal.pone.0061217>.

Description

Transform 'microtable' object of 'microeco' package to the 'phyloseq' object of 'phyloseq' package <doi:10.1371/journal.pone.0061217>.

Usage

meco2phyloseq(meco)

Arguments

meco

a microtable object.

Value

phyloseq object.
