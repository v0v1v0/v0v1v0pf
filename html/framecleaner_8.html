date_ym R Documentation

date_ym

Description

creates a monthly date vector from a date vector

Usage

date_ym(x)

Arguments

x

a date

Value

date vector

Examples


seq.Date(lubridate::ymd(20200101), lubridate::ymd(20220101), length.out = 10) -> d1
d1 %>%
  tibble::enframe() %>%
  dplyr::mutate(YM = date_ym(value))
