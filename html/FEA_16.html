beamGlobalEM R Documentation

beamGlobalEM

Description

Generates global stiffness matrix for beams.

Usage

beamGlobalEM(beamExEM)

Arguments

beamExEM

Expanded Element Matrix

Value

Produces large (n x n) global matrix

GlobalMat

Global matrix

Examples

data(beamExMat)

beamExEM = beamExMat
GMat = beamGlobalEM(beamExEM)
