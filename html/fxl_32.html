LozyEtAl2020 R Documentation

Plotting data from Lozy et al. (2020)

Description

Treatment data from Lozy et al. (2020)

Usage

LozyEtAl2020

Format

A data frame with 91 rows and 5 variables:

Session

Session number

Participant

Participant name

KM

Kinesthetic movement choices

TD

Traditional drill choices

Phase

Sequenced phase number

Source
