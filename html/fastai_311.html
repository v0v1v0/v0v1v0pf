get_files R Documentation

Get_files

Description

Get all the files in 'path' with optional 'extensions', optionally with 'recurse', only in 'folders', if specified.

Usage

get_files(
  path,
  extensions = NULL,
  recurse = TRUE,
  folders = NULL,
  followlinks = TRUE
)
