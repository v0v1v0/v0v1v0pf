fslview.help R Documentation

FSLView help

Description

This function calls fslview's help

Usage

fslview.help()

Value

Prints help output and returns output as character vector

Examples

library(fslr)
if (have.fsl()){
  print(fsl_version())
  in_ci <- function() {
   nzchar(Sys.getenv("CI"))
  }
 if (!in_ci()) {
   fslview.help()
 }
}
