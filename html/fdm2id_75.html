general.rules R Documentation

Remove redundancy in a set of rules

Description

This function remove every redundant rules, keeping only the most general ones.

Usage

general.rules(r)

Arguments

r

A set of rules.

Value

A set of rules, without redundancy.

See Also

apriori

Examples

require ("arules")
data ("Adult")
r = apriori (Adult)
inspect (general.rules (r))
