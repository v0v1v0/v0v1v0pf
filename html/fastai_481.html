InstanceNorm R Documentation

InstanceNorm

Description

InstanceNorm layer with 'nf' features and 'ndim' initialized depending on 'norm_type'.

Usage

InstanceNorm(
  nf,
  ndim = 2,
  norm_type = 5,
  affine = TRUE,
  eps = 1e-05,
  momentum = 0.1,
  track_running_stats = FALSE
)

Arguments

nf

input shape
ndim

dimension number
norm_type

normalization type
affine

affine
eps

epsilon
momentum

momentum
track_running_stats

track running statistics

Value

None
