DF1 R Documentation

1DF Catalog (First D3PO Fermi catalog of gamma-ray source candidates)

Description

The Denoised, Deconvolved, and Decomposed Fermi Gamma-ray Sky: An application of the D3PO algorithm Selig, M. et al., Astronomy and Astrophysics, 581, 126 (2015).

Usage

DF1

Format

A data frame with 48 variables on 3106 gamma-ray sources.

Details

FITS Filename: catalog_1.fits
