multi.argument.Compose R Documentation

Composition with multiple arguments.

Description

Thanks, Alexander Davis!

Usage

multi.argument.Compose(...)

Arguments

...

the functions to be composed

Value

A composed function

Examples

f <- function(x, y) x+y
g <- function(x) x*2
stopifnot(multi.argument.Compose(f, g)(1,1) == 4)
