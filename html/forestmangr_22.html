exfm18 R Documentation

Experimental data on nitrogen effect in species fertilizing

Description

In this data, each observation is a tree.

Usage

data(exfm18)

Format

A data frame with 877 observations and 6 variables:

Plot

plot number

Species

species scientific name

Tree

tree number

Trunk

trunk number

CBH

circumference at breast height, in centimeters

DBH

diameter at breast height, in centimeters
