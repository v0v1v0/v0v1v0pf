fuzzy_and R Documentation

Fuzzy And Function

Description

Fuzzy And Function

Usage

fuzzy_and(inputA, inputB)

Arguments

inputA

First input vector
inputB

Second input vector. Must be of the same dimension as inputA.

Value

Returns the Fuzzy AND of two input values in a vector.

Examples

fuzzy_and(0, -1) # = -1
fuzzy_and(0, 1) # = 0
fuzzy_and(1, 2) # = 1
fuzzy_and(1, 1) # = 1
fuzzy_and(c(0.5, 0.75), c(1.5, 1)) # = c(0.5,0.75)
