chapman R Documentation

Chapman data

Description

The chapman data frame has 200 rows and 7 columns.

Usage

data(chapman)

Format

This data frame contains the following columns:

age

a numeric vector

highbp

a numeric vector

lowbp

a numeric vector

chol

a numeric vector

height

a numeric vector

weight

a numeric vector

y

a numeric vector
