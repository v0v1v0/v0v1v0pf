line_spacing R Documentation

Set text alignment

Description

change text alignment of selected rows and columns of a flextable.

Usage

line_spacing(x, i = NULL, j = NULL, space = 1, part = "body")

Arguments

x

a flextable object
i

rows selection
j

columns selection
space

space between lines of text, 1 is single line spacing, 2 is double line spacing.
part

partname of the table (one of 'all', 'body', 'header', 'footer')

See Also

Other sugar functions for table style: align(), bg(), bold(), color(), empty_blanks(), font(), fontsize(), highlight(), italic(), keep_with_next(), padding(), rotate(), tab_settings(), valign()

Examples

ft <- flextable(head(mtcars)[, 3:6])
ft <- line_spacing(ft, space = 1.6, part = "all")
ft <- set_table_properties(ft, layout = "autofit")
ft
