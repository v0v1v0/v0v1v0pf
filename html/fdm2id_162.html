SVD R Documentation

Singular Value Decomposition

Description

Return the SVD decomposition.

Usage

SVD(x, ndim = min(nrow(x), ncol(x)), ...)

Arguments

x

A numeric dataset (data.frame or matrix).
ndim

The number of dimensions.
...

Other parameters.

See Also

svd

Examples

require (datasets)
data (iris)
SVD (iris [, -5])
