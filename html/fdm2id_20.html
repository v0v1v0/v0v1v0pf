cartplot R Documentation

CART Plot

Description

Plot a decision tree obtained by CART.

Usage

cartplot(model, ...)

Arguments

model

The decision tree.
...

Other parameters.

See Also

CART, cartdepth, cartinfo, cartleafs, cartnodes

Examples

require (datasets)
data (iris)
model = CART (iris [, -5], iris [, 5])
cartplot (model)
