fslbet.help R Documentation

Help for FSL BET

Description

This function calls bet's help

Usage

fslbet.help(betcmd = c("bet2", "bet"))

Arguments

betcmd

(character) Get help for bet or bet2 function

Value

Prints help output and returns output as character vector

Examples

if (have.fsl()){
 fslbet.help()
 fslbet.help("bet")
}
