Gilroyetal2021 R Documentation

Plotting data from Gilroy et al. (2015) - Treatment

Description

Treatment data from Gilroy et al. (2021)

Usage

Gilroyetal2021

Format

A data frame with 69 rows and 7 variables:

Participant

Participant name

Session

Session number

Condition

Functional analysis condition

Responding

Rates of responding

Reinforcers

Reinforcer deliveries

PhaseNum

Sequenced phase number

LineOff

Offset of phase line

Source

<doi:https://doi.org/10.1002/jaba.826>
