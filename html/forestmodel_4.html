forest_model_format_options R Documentation

Create format options for forest_model

Description

Create format options for forest_model

Usage

forest_model_format_options(
  colour = "black",
  color = NULL,
  shape = 15,
  text_size = 5,
  point_size = 5,
  banded = TRUE
)

Arguments

colour

colour of the point estimate and error bars
color

alias for colour
shape

shape of the point estimate
text_size

text size in mm
point_size

point size
banded

whether to show light grey bands behind alternate rows

Value

list of format options
