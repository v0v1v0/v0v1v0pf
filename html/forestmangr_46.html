na_to_0 R Documentation

Convert NA to zero on numeric variables

Description

This function Convert NA to zero on numeric variables outside of mutate. this is used because of dplyr 1.1.0.

Usage

na_to_0(df)

Arguments

df

A data frame

Value

a data frame
