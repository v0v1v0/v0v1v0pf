bliss R Documentation

Bliss data

Description

The bliss data frame has 8 rows and 4 columns.

Usage

data(bliss)

Format

This data frame contains the following columns:

Dose

a numeric vector

Killed

a numeric vector

Total

a numeric vector

y

a numeric vector
