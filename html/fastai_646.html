PixelShuffle_ICNR

Description

Upsample by 'scale' from 'ni' filters to 'nf' (default 'ni'), using 'nn.PixelShuffle'.

Usage

PixelShuffle_ICNR(
  ni,
  nf = NULL,
  scale = 2,
  blur = FALSE,
  norm_type = 3,
  act_cls = nn()$ReLU
)

Arguments

ni

input shape
nf

number of features / outputs
scale

scale
blur

blur
norm_type

normalziation type
act_cls

activation

Value

None
