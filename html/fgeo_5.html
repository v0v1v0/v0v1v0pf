fgeo_browse R Documentation

Open a web browser on fgeo's website.

Description

Load fgeo's URLs into an HTML browser.

Usage

fgeo_browse()

fgeo_browse_reference()

See Also

Other functions to explore fgeo: fgeo_help, fgeo_packages

Examples

if (interactive()) {
  fgeo_browse()
  fgeo_browse_reference()
}
