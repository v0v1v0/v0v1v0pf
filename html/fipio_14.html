fips_county R Documentation

Get the county name for a FIPS code

Description

Get the county name for a FIPS code

Usage

fips_county(fip)

Arguments

fip

2-digit or 5-digit FIPS code

Value

a character vector

Examples

fipio::fips_county("37129")
fipio::fips_county("06001")

# 2-digit FIP codes will not work
fipio::fips_county("37")
