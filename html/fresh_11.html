bs4dash_font R Documentation

bs4Dash fonts parameters

Description

bs4Dash fonts parameters

Usage

bs4dash_font(
  size_base = NULL,
  size_lg = NULL,
  size_sm = NULL,
  size_xs = NULL,
  size_xl = NULL,
  weight_light = NULL,
  weight_normal = NULL,
  weight_bold = NULL,
  family_sans_serif = NULL,
  family_monospace = NULL,
  family_base = NULL
)

Arguments

size_base

Base size, this size is used to calculate other size. Must in rem unit.
size_lg

Large size.
size_sm

Small size.
size_xs

Extra small size.
size_xl

Extra large size.
weight_light

Light font weight.
weight_normal

Normal font weight.
weight_bold

Bold font weight.
family_sans_serif

Sans serif font family.
family_monospace

Monospace font family.
family_base

Base font family.

Value

a list that can be used in create_theme.

Examples


# Change font size used in bs4Dash
bs4dash_font(
  size_base = "1.5rem",
  weight_bold = 900
)
