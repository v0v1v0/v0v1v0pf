logit R Documentation

Logit transformation

Description

Computes the logit transformation

Usage

logit(x)

Arguments

x

a numeric vector

Details

x <=0 or >=1 will return NA

Value

log(x/(1-x))

Author(s)

Julian Faraway

See Also

ilogit

Examples

logit(c(0.1,0.5,1.0,1.1))
#[1] -2.197225  0.000000        NA        NA
