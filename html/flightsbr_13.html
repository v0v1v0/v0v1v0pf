get_airfares_url R Documentation

Put together the url of airfare data files

Description

Put together the url of airfare data files

Usage

get_airfares_url(dom, year, month)

Arguments

dom

Logical. Defaults to TRUE download airfares of domestic flights. If FALSE, the function downloads airfares of international flights.
year

Numeric. Year of the data in yyyy format.
month

Numeric. Month of the data in mm format.

Value

A url string.

Examples

## Not run:  if (interactive()) {
# Generate url
a <- get_airfares_url(year=2002, month=11)
}
## End(Not run)
