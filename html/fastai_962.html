Warp

Description

Apply perspective warping with 'magnitude' and 'p' on a batch of matrices

Usage

Warp(
  magnitude = 0.2,
  p = 0.5,
  draw_x = NULL,
  draw_y = NULL,
  size = NULL,
  mode = "bilinear",
  pad_mode = "reflection",
  batch = FALSE,
  align_corners = TRUE
)

magnitude

magnitude
p

probability
draw_x

draw x

draw y

mode
