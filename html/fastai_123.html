cnn_config R Documentation

Cnn config

Description

Convenience function to easily create a config for 'create_cnn_model'

Usage

cnn_config(
  cut = NULL,
  pretrained = TRUE,
  n_in = 3,
  init = nn()$init$kaiming_normal_,
  custom_head = NULL,
  concat_pool = TRUE,
  lin_ftrs = NULL,
  ps = 0.5,
  bn_final = FALSE,
  lin_first = FALSE,
  y_range = NULL
)

Arguments

cut

cut
pretrained

pre-trained or not
n_in

input shape
init

initializer
custom_head

custom head
concat_pool

concatenate pooling
lin_ftrs

linear filters
ps

parameter server
bn_final

batch normalization final
lin_first

linear first
y_range

y_range

Value

None
