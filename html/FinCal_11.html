Description

Computing the rate of return for each period

Usage

discount.rate(n, pv, fv, pmt, type = 0)

Arguments

n

number of periods
pv

present value
fv

future value
pmt

payment per period
type

payments occur at the end of each period (type=0); payments occur at the beginning of each period (type=1)
