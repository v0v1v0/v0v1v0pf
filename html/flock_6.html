unlock R Documentation

Locking/Unlocking

Description

Unlocks a file previously locked via lock.

Usage

unlock(file.lock)

Arguments

file.lock

The FileLock object returned by lock.

Author(s)

Ivan Popivanov

See Also

lock

Examples

## Not run: 
require(flock)

file.lock = lock("~/file.lock")
# Critical section code goes here
unlock(file.lock)

## End(Not run)
