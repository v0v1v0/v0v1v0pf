halfnorm R Documentation

Half Normal Plot

Description

Makes a half-normal plot

Usage

halfnorm(
  x,
  nlab = 2,
  labs = as.character(1:length(x)),
  ylab = "Sorted Data",
  ...
)

Arguments

x

a numeric vector
nlab

number of points to label
labs

labels for points
ylab

label for Y-axis
...

arguments passed to plot()

Value

none

Author(s)

Julian Faraway

See Also

qqnorm

Examples


halfnorm(runif(10))
