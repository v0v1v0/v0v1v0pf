cartnodes R Documentation

Number of Nodes

Description

Return the number of nodes of a decision tree.

Usage

cartnodes(model)

Arguments

model

The decision tree.

Value

The number of nodes.

See Also

CART, cartdepth, cartinfo, cartleafs, cartplot

Examples

require (datasets)
data (iris)
model = CART (iris [, -5], iris [, 5])
cartnodes (model)
