form_deciles R Documentation

Form deciles

Description

Calculate deciles for a variable.

Usage

form_deciles(x)

Arguments

x

A vector for which deciles are to be calculated.

Value

vector

Examples

library(farr)
library(dplyr, warn.conflicts = FALSE)

df <-
    tibble(x = rnorm(100)) %>%
    mutate(dec_x = form_deciles(x))
df
