Description

Allow autoplotting the columns sp and elev.

Usage

sp_elev(sp, elev = NULL)

Arguments

sp

A ForestGEO-like dataframe with the column sp.
elev

A ForestGEO-like elevation list or its col dataframe â€“ with the column elev. The datasets you pass to sp and elev should come from the same forest plot. This is not compulsory but not doing so is most likely a mistake.

Value

An S3 object of class 'sp_elev'.

Examples

species_from_luquillo <- fgeo.x::stem5
elevation_from_luquillo <- fgeo.x::elevation
