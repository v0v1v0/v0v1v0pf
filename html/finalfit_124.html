round_tidy R Documentation

Round values but keep trailing zeros

Description

e.g. for 3 decimal places I want 1.200, not 1.2.

Usage

round_tidy(x, digits)

Arguments

x

Numeric vector of values to round
digits

Integer of length one: value to round to.

Value

Vector of strings.

Examples

round_tidy(0.01023, 3)
