fine_tune R Documentation

Fine_tune

Description

Fine tune with 'freeze' for 'freeze_epochs' then with 'unfreeze' from 'epochs' using discriminative LR

Usage

fine_tune(
  object,
  epochs,
  base_lr = 0.002,
  freeze_epochs = 1,
  lr_mult = 100,
  pct_start = 0.3,
  div = 5,
  ...
)

Arguments

object

learner/model
epochs

epoch number
base_lr

base learning rate
freeze_epochs

freeze epochs number
lr_mult

learning rate multiply
pct_start

start percentage
div

divide
...

additional arguments

Value

None
