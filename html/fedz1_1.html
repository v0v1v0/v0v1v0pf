all_tables_title R Documentation

Title of tables

Description

all_tables_title() returns title of all tables. It is helpful to get familiar with available tables. Each table contains a number of data set.

Usage

all_tables_title()

Value

A tibble of title of all tables

See Also

https://www.federalreserve.gov/apps/fof/FOFTables.aspx

Examples

all_tables_title()
