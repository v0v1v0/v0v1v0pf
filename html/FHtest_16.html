Function ics

Description

Internal function

Usage

  ics(icFIT, group, rho, alternative = "two.sided", tol.svd = 10^-8)

Arguments

icFIT

Object of type icFIT
group

Grouping variable
rho

A scalar parameter that controls the type of test.
alternative

Character giving the type of alternative hypothesis for two-sample and trend tests.
tol.svd

Tolerance to determine convergence.
