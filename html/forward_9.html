dialectric R Documentation

Dialectric data

Description

The dialectric data frame has 128 rows and 3 columns.

Usage

data(dialectric)

Format

This data frame contains the following columns:

time

Time (weeks)

temp

Temperature (degrees Celsius)

y

dialectric breakdown strength in kilovolts
