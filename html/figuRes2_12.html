default.settings

Description

Global Defaults

Usage

default.settings(
  pos = 1,
  envir = as.environment(pos),
  my.path = getwd(),
  main.theme = "theme_bw",
  page.width = 11,
  page.height = 8.5,
  right.margin = 0.75,
  left.margin = 0.75,
  top.margin = 1.4 - 0.5,
  bottom.margin = 1.75 - 0.5
)

used internally by some functions

Details

Global Defaults

Value

This function assigns character string objects to the global environment.

Value

The following are assigned to global environment upon calling:

  • my.path

  • dd

  • cd

  • od

  • blankPanel

  • page.width

  • page.height

  • right.margin

  • left.margin

  • top.margin

  • bottom.margin

  • graph.region.h

  • graph.region.w
