write2_desc R Documentation

Data Writing Functions

Description

These functions allow for data to be written to different formats. Currently, supported formats are "csv" and "sqlite".

Usage

write2csv(
  layer,
  outfile = NULL,
  outfolder = NULL,
  outfilenm = NULL,
  outfn.pre = NULL,
  outfn.date = FALSE,
  overwrite = FALSE,
  tabtitle = NULL,
  appendfile = FALSE,
  closefn = TRUE,
  outtxt = NULL,
  gui = FALSE
)

write2sqlite(
  layer,
  SQLitefn = NULL,
  out_name = NULL,
  gpkg = FALSE,
  outfolder = NULL,
  overwrite = FALSE,
  append_layer = FALSE,
  createnew = FALSE,
  dbconn = NULL,
  dbconnopen = FALSE,
  index.unique = NULL,
  index = NULL
)

Details

These functions allow for data to be written to different formats.

Value

No return value, writes data to either a "csv" or "sqlite" file.
