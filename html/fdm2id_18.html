cartleafs R Documentation

Number of Leafs

Description

Return the number of leafs of a decision tree.

Usage

cartleafs(model)

Arguments

model

The decision tree.

Value

The number of leafs.

See Also

CART, cartdepth, cartinfo, cartnodes, cartplot

Examples

require (datasets)
data (iris)
model = CART (iris [, -5], iris [, 5])
cartleafs (model)
