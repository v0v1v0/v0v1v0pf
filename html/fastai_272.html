Flip R Documentation

Flip

Description

Randomly flip a batch of images with a probability 'p'

Usage

Flip(
  p = 0.5,
  draw = NULL,
  size = NULL,
  mode = "bilinear",
  pad_mode = "reflection",
  align_corners = TRUE,
  batch = FALSE
)

Arguments

p

probability
draw

draw
size

size of image
mode

mode
pad_mode

reflection, zeros, border as string parameter
align_corners

align corners ot not
batch

batch or not

Value

None
