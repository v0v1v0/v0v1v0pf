Hook R Documentation

Hook

Description

Create a hook on 'm' with 'hook_func'.

Usage

Hook(
  m,
  hook_func,
  is_forward = TRUE,
  detach = TRUE,
  cpu = FALSE,
  gather = FALSE
)

Arguments

m

m aprameter
hook_func

hook function
is_forward

is_forward or not
detach

detach or not
cpu

cpu or not
gather

gather or not

Details

Hooks are functions you can attach to a particular layer in your model and that will be executed in the forward pass (for forward hooks) or backward pass (for backward hooks).

Value

None
