Description

Glance an ATA model

Usage

## S3 method for class 'ATA'
glance(x, ...)

Arguments

x

An estimated model.
...

Unused.

Value

A one row tibble summarising the model's fit.

Examples

library(fable.ata)
as_tsibble(USAccDeaths) %>%
  model(ata = AutoATA(value ~ trend("A") + season("M"))) %>% glance()
